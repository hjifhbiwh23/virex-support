💾 Step 1: Preparing USB

Create a Windows installation USB and prepare all required files.

// Create Installation Media
01

Download the Media Creation Tool from microsoft.com and click Download Now.

02

Open the Media Creation Tool and select Create installation media (USB flash drive, DVD or ISO file) for another PC.

03

Use the recommended options and select USB flash drive (minimum 8 GB). Select your USB — all data will be removed.

04

Download the Autounattended.xml file and drag it together with the Windows files onto your USB.

// Getting Ethernet Drivers
05

Open Device Manager → Network Adapters and identify your Ethernet adapter type.

06

Download the correct driver:

• Realtek Ethernet: Download Realtek Driver

Extract the downloaded drivers onto your USB stick. You'll need them later during the reinstall.

// Prepare BIOS Flash
⚠️ Warning: Incorrect BIOS flashing can damage your system. Ensure you have the correct BIOS version and a stable power source. Ask support if unsure!
07

Open System Information (Win + R → msinfo32). Look for BaseBoard Product and BaseBoard Manufacturer under System Summary.

08

To enter BIOS directly, open CMD as Administrator and run:

shutdown /r /fw /t 0

09

Search for your motherboard model on the manufacturer's website and go to Downloads / Support → BIOS.

  • ASUS: Downgrade to a version PRE 2022. If already at the lowest, upgrade by only one version.
  • Run BIOS Renamer.exe first, then extract the .CAP file to your USB.
10

Your USB should now contain: Windows files, Autounattended.xml, Ethernet drivers, and the BIOS file.

Wait for the BIOS update to finish. The PC will reboot automatically. Do not power off during flashing.

⚙️ Step 3: Configuring BIOS

Disable the required features in BIOS for the spoofer to work correctly.

ℹ️ Every motherboard is different. If you can't find the right option, open a support ticket.
// Disable the following (Intel)
ASUS / MSI / GIGABYTE / Other — location varies per board. Search for these by name.
  • Virtualization — Disable
  • VT-D — Disable
  • TPM — Disable
  • Utility Installer — Disable
  • Fast Boot — Disable
  • Secure Boot — Disable (set OS Type to Other OS)
  • Wifi & Bluetooth — Disable (WAN Radio on ASRock)
// Disable the following (AMD — ASUS example)
  • SVM Mode (Virtualization): Advanced → CPU Configuration → SVM Mode → Disabled
  • AMD NX Mode: Advanced → CPU Configuration → NX Mode → Disabled
  • AMD IOMMU: Advanced → AMD CBS → IOMMU → Disabled
  • AMD fTPM: Advanced → AMD fTPM Switch → Disabled
  • Security Device Support: Advanced → Trusted Computing → Disabled
  • Armoury Crate: Tool → ASUS Armoury Crate → Download & Install → Disabled
  • Fast Boot: Boot → Boot Configuration → Fast Boot → Disabled
  • Secure Boot: Boot → Secure Boot → OS Type → Other OS
  • Wifi & Bluetooth: Advanced → Onboard Devices → Disabled

🖥️ Step 4: Special Windows Reinstall

Install Windows using RAID mode for permanent hardware ID spoofing.

ℹ️ First run the Compatibility Checker to determine if your system is RAID0 / RAIDABLE compatible. If not compatible, open a ticket.
// AMD — Download Drivers
📦 AM4 RAID Files.zip 📦 AM5 RAID Files.zip

Check your CPU in Task Manager → Performance → CPU. Extract the drivers to your USB stick.

// Enable RAID Mode in BIOS
  • Restart and enter BIOS. Find SATA Mode: ASUS: Advanced→SATA Config / Gigabyte: Settings→IO Ports / ASRock: Advanced→Storage Config / MSI: Advanced→Integrated Peripheral
  • Set SATA mode/controller → RAID mode (change from AHCI)
  • Set NVMe mode → Enabled
  • Boot tab: Set CSM → Disabled
  • Save & exit, then re-enter BIOS!
// Clear Disks — RAIDXpert2
  • In BIOS: Advanced → RAIDXpert2 Configuration Utility (Gigabyte: Settings→IO Ports→RAIDXpert2)
  • Enter Manage Arrays → Delete Arrays — select every disk and confirm delete (all data will be lost)
// Create RAIDABLE Array
  • RAIDXpert2 → Manage Arrays → Create Array
  • RAID Level: Raidable
  • Select Physical Disks → Enable → Apply Changes → Create Array
  • If you have multiple disks: create one array per disk
// Installing Windows (AM4)
  • Save & exit BIOS, boot from USB
  • At "Where do you want to install Windows?" → click Load driver → Browse → navigate to USB → select NVMe_RAID or SATA_RAID folder → OK
  • Choose "AMD-RAID Bottom Device" driver → Next
  • Click Load driver again → same folder → choose "AMD-RAID Controller" driver → Next
  • Select disk → install Windows
✅ To verify: Task Manager → Performance → Disk 0 (C:) → should say AMD-RAID
// Enable RAID Mode in BIOS (Intel)
Look for: SATA Mode + M.2 Storage Raid Support. Common locations: PCH Storage Config, Storage Config, VMD Controller, or SA System Agent.
  • If VMD Controller option exists → enable it (ignore SATA mode below)
  • Set SATA Mode/controller → RAID mode / RST Controlled / Optane Mode
  • Set M.2_1 RST PCIE Storage Raid Support → Enabled / RST Controlled
  • Set M.2_2 RST PCIE Storage Raid Support → Enabled / RST Controlled
  • Boot tab: Set CSM → Disabled — Save & exit, re-enter BIOS
// Create RAID0 Array (Intel RST)
  • Advanced → Intel(R) RST → Create RAID Volume
  • Set RAID Level → RAID0 (Stripe)
  • Select your disks → Create Volume
// Installing Windows
  • Save & exit BIOS, boot from USB
  • At "Where do you want to install Windows?" → Load driver → Browse → navigate to drivers folder → OK → Ctrl+A to select all → Next
  • Select disk and install Windows
✅ To verify: Task Manager → Performance → Disk 0 (C:) → should say INTEL RAID
❌ Not showing as INTEL RAID 0 VOLUME? Open a ticket in the Discord server immediately.

🌐 Step 9: Ethernet Settings

Configure your network adapter to reduce detection vectors.

01

Press Win + R, type ncpa.cpl, and press Enter to open Network Connections.

02

Right-click your Ethernet connection → Properties. Make sure only these two options are checked (disable all others):

  • ✅ QoS Packet Scheduler
  • ✅ Internet Protocol Version 4 (TCP/IPv4)

03

Right-click Ethernet → Properties → Configure → Advanced tab. Apply these settings:

  • Advanced EEE → Disabled
  • ARP Offload → Disabled
  • Flow Control → Disabled
  • IPv4 Checksum Offload → Disabled
  • Large Send Offload v2 (IPv6) → Disabled
  • Network Address → Not Present
  • TCP Checksum Offload (IPv6) → Disabled
  • UDP Checksum Offload (IPv6) → Disabled

If any option is not available, skip it.

04

Click OK and restart your PC.

05

After restarting, download and run networksettings.bat as Administrator. Follow the on-screen instructions.

06

VPN Setup: Download and install Cloudflare WARP (1.1.1.1). Select 1.1.1.1 with WARP and enable it.
