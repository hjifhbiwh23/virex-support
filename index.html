DOWNLOAD LOADER
// SETUP PROCEDURE
01

If you use the cheat for the first time, you have to register first. The register button is right below the login button — click it.

02

Enter your desired username and password. Make sure to meet all password requirements before continuing.

03

Type in your Invite Code and Cheat Key as provided after purchase. Both are required to complete registration.

04

After registration succeeds, log in with your credentials. You should now see your active cheat subscriptions in the dashboard.

05

Press the Load button. A pop-up will appear asking you to reboot your machine. Confirm and restart your PC.

06

After rebooting, re-open the loader and log in again with your credentials.

07

Press Load for the second time. If everything went correctly, the loader will close itself automatically. This is normal behavior — if the loader stays open, something went wrong.

08

Start Rainbow Six Siege and make sure the game is running in BORDERLESS window mode. This is required for the overlay to work.

09

Once you are inside a game window, press the INSERT key — the menu should appear. Use INSERT to toggle the menu open and closed at any time.

// Important Notes

  • Always run the loader as Administrator for best results.
  • If injection fails, make sure your antivirus / Windows Defender is disabled.
  • The loader closing itself after the second Load is expected — do not reopen it.
  • Use BORDERLESS window mode only. Fullscreen and Windowed modes are not supported.
  • If you encounter issues, open a support ticket — do not share keys or credentials.
